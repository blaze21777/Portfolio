The treasure hunt was a project I completed for my Certificate in digital applications coursework. I was tasked to create and interactive treasure hunt and then create a website using Dreamweaver to present my methodology.

This website was created to showcase how a project from Dreamweaver was changed to one using HTML, CSS and JavaScript.

During this project I have learnt how to use frameworks like bootstrap to build the foundation of a webpage then use my own knowledge to build upon this and customize the template to make it my own.
